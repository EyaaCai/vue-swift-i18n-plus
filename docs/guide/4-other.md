## 扁平/反扁平 json

在开始做国际化时候，未扁平的 i18n json 需要一层层复制，扁平/反扁平 json 可以减少复制次数的同时，让 i18njson 更加可读。

## 代码提示

在复制完 key 之后，要使用对应的语法去做国际化，这里提供了三条代码提示

1. **`t`**  替换为 `<b>`&#36;t('剪切板内容')`</b>`，需手动加 `：`
2. **`tt`** 替换为 `<b>`&#36;t('剪切板内容')`</b>`
3. **`ttt`** 替换为 `<b>`this.&#36;t('剪切板内容')`</b>`
